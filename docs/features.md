# CaliAI – Feature Specification

This document describes the functional capabilities of the system.

---

# 1. Core Feature – AI Form Analysis

The primary feature of CaliAI.

## 1.1 Video Upload

- User records or uploads exercise video.
- Supported exercises (MVP):
  - Push-ups
  - Pull-ups
  - Squats
  - Static holds (e.g. plank)

## 1.2 AI Analysis

The system analyzes:

- Body posture
- Joint alignment
- Range of motion
- Stability
- Symmetry

## 1.3 Feedback Report

After analysis, the user receives:

- Form score (0–100)
- Identified mistakes
- Suggested corrections
- Visual indicators (future enhancement)
- Historical comparison
